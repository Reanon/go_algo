# 数据结构与算法

![img](https://aliyun-typora-img.oss-cn-beijing.aliyuncs.com/imgs/20210115150349.png)



> 10 个数据结构

基础数据结构：数组、链表、栈、队列

进阶数据结构：散列表、二叉树、堆、跳表、图、Trie 树

> 10 个算法

基础算法：递归、排序、二分查找、搜索

进阶算法：哈希算法、贪心算法、分治算法、回溯算法、动态规划、字符串匹配算法

> [!note]
>
> 学习数据结构的要点

- 来历
- 自身的特点
- 适合解决的问题
- 实际的应用场景
